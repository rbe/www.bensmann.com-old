Im Fokus stehen dabei stets zwei Aspekte: To Code and To Serve

To Code

Die Schwerpunkte meiner Tätigkeit liegen hier auf den folgenden Bereichen:

  • auf der Architektur und Entwicklung von Konzepten und Software, basierend auf den Technologien:
    Java Standard und Enterprise Edition, Groovy & Grails, Clojure,
    OpenOffice,
    Oracle Datenbanken und Real Application Clusters
    Solaris und Linux
  • auf meiner Tätigkeit als Trainer und Ausbilder:
    Hauptsächlich für Java-Kurse bei Schulungspartner von Oracle sowie als Ausbilder für Fachinformatiker mit der Fachrichtung Anwendungsentwicklung (IHK)

    Gemeinsam mit meinen Partnern
    auf der Basis von Solaris und Linux,
    Oracle Datenbanken und RAC

... and To Serve

Die langjährige Erfahrung in der Entwicklung und Architektur von Software sowie im Betrieb von IT-Systemen erlaubt es mir, meine Fähigkeiten, Kenntnisse und Kompetenzen mit dem notwendigen Gespür für individuelle Unternehmensbedürfnisse punktgenau einzusetzen und so Ihre Projekte zum Erfolg zu führen.

Zu meinen Kunden gehören kleine, mittelständische, aber auch große Unternehmen, u. a. aus den Branchen:

  • Gesundheitswesen
  • Industrie
  • Informationstechnologie
  • Verlagswesen
